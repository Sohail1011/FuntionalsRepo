¿Qué es una función?

Concepto

Sirver para usar el mismo proceso, pero con parametros para las variables en su ejecución.

Tipos de funciones

Hay dos tipos de funciones:

  • Declaración

    • Sirve como una simple función normal con parametros a su costado, entre parentesís. Y abre llaves, para dar instrucciones.

  • Expresion

    • Se declara con una variable, seguido los parentesís, y el '=>' junto de las llaves para dar instrucciones.

Como se ejecuta la función en Javascript

Se pone el nombre de la función, después se pone los parentesís al final.
